# Editar Archivo Hosts en Windows
``` 
* Presionar tecla windows 
* Escibir notepad
* Presionar Boton secundario sobre el icono
* En el menu desplegable presionar "Ejecutar como administrador"
* Seleccionar Archivo -> Abrir
* Buscar el archivo "hosts" en la siguiente ruta "C:\Windows\System32\drivers\etc"
* Al final del archivo escribir lo siguiente "127.0.0.1 magento.test" (Sin las comillas)
* Guardar
```
